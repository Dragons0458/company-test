## Description

This is a test for a company.

## Running with Docker compose

```bash
$ docker-compose up -d
```

## Deleting and stopping the containers

```bash
$ docker-compose down
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
